Initialize maven:
mvn archetype:generate -DgroupId=com.haingue.check-list -DartifactId=check-list -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

Build Project:
mvn package

Build test:
mvn test

Compile Project:
mvn compile

Generate Site:
mvn site

Exemple:
mvn clean compile testpackage
